Signature draggable before/after comparison slider — the hero device for a painting brand's gallery and service pages. Renders labeled placeholder blocks until the client supplies photos.

```jsx
<BeforeAfter beforeLabel="Before" afterLabel="After" height={460} />
<BeforeAfter before="/before.jpg" after="/after.jpg" alt="Exterior repaint in Jupiter, FL" />
```

Drag the amber handle (mouse or touch) to wipe between images. Amber divider + circular handle.

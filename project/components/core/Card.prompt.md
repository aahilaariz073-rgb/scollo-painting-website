Generic surface container — white with hairline warm border + soft shadow. Set `hover` for interactive cards that lift.

```jsx
<Card hover>…</Card>
<Card variant="navy">…</Card>   {/* inverse, for navy bands */}
<Card variant="panel">…</Card>  {/* subtle warm fill */}
```

Variants: `default`, `flat` (no shadow), `panel` (warm fill), `navy` (inverse). Radius `lg` (14px).

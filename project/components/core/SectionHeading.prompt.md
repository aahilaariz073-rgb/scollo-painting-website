Editorial section header — gold rule, uppercase eyebrow, serif title with optional italic accent, and a lead paragraph. Opens nearly every section.

```jsx
<SectionHeading
  eyebrow="Why homeowners choose us"
  title="Done right,"
  accent="once."
  lead="Clean lines, durable finishes, and pricing quoted up front."
/>
```

Use `tone="light"` on navy bands, `align="center"` for centered intros, `as="h1"` for the page H1.
